# 🌾 Prediction of Agricultural Crop Production in India

## 📌 Project Overview

This project predicts agricultural crop production in India using Machine Learning. Historical agricultural datasets were analyzed to identify important factors affecting crop production such as crop type, variety, cultivation cost, state, quantity, season, and recommended cultivation zone.

The project applies data preprocessing, exploratory data analysis (EDA), feature engineering, and multiple machine learning algorithms to accurately predict crop production.

---

## 🎯 Objectives

- Analyze agricultural production data
- Clean and preprocess datasets
- Perform Exploratory Data Analysis (EDA)
- Train Machine Learning models
- Compare model performance
- Predict crop production
- Save the trained model

---

| Feature | Description |
